### Project Installation

1. Clone the repository

```git clone https://github.com/darksos34/Flightdata.git ```

2. Go into the ```flightdata``` folder:

```cd flightdata ```

3. Build the project using Maven:

```mvn clean install ```

4. Run the Spring boot application:

```mvn spring-boot:run ```
